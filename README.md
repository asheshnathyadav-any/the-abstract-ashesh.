## 🛠️ Built With & Concepts Applied

This portfolio was designed from the ground up without heavy external frontend frameworks to establish a rock-solid understanding of web fundamentals:

*   **Semantic HTML5:** Structured cleanly using native structural elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` to optimize SEO and accessibility.
*   **CSS Grid (2D Layouts):** Implemented an automated grid system to handle multi-row asset displays cleanly (such as the 3x3 logo layout) without manual line breaks.
*   **CSS Flexbox (1D Layouts):** Leveraged Flexbox to create a fluid, sticky navigation menu bar and centralized profile action link elements.
*   **Fluid Responsiveness:** Programmed media queries that dynamically adapt layouts across mobile, tablet, and widescreen desktop formats.
